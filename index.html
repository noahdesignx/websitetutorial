HOW TO BUILD A WEBSITE
(HTML & CSS)

OVERVIEW


Websites are a great way to showcase your professional career or business. While creating a website is easier than ever before with all of the web development tools available to us, many of these tools limit you from customizing your website the way you want it to look like. The best way to avoid these restrictions is by programming the website yourself from scratch. Although it may initially sound intimidating, web development programming languages are really easy to learn. In this tutorial, you will learn how to use basic HTML and CSS to develop a photographer portfolio at a beginner level and host it online for free.




Web Development Languages:


Hypertext Markup Language (HTML) is a markup language that acts as the skeleton or basic framework for a website.


Cascading Style Sheets (CSS) is a styling sheet language that decorates and formats the skeleton or basic framework of a website.


JavaScript (JS) is a programming language that gives a website more advanced functionality, but for this tutorial, this will not be needed.

Set Up Replit


You will need to create an account in Replit so that you have free access to an integrated development environment (IDE).

1. Go to replit.com

2. On the home page, click Sign Up For Free

3. Follow the instructions to finish creating your account

2

Create and Manage Project Files


You will create your main project file and upload the images needed for the website.

1. On your home dashboard, click Create Repl

2. Select the template HTML, CSS, JS

3. Name your project

4. Click Create

5. In the file panel to the left, click on the three dots and then Upload Folder to upload the images folder that you were instructed to download earlier

3

Set Website Name and Favicon


In the index.html file, you will update the name and icon of the website that shows in the browser tab. These changes will be made within the header tag.

1. In the title tag, name the website Michelle Flores <title>Michelle Flores</title>

2. Below the title tag, add <link rel = "icon" href = "images/logo.png"> to set the favicon of the website to the logo image file

4

Create Framework of Home Section


In the index.html file, you will create the framework of the home section of the website where the user can read the job title, navigate to different parts of the website, and contact their email address.

1. In the body tag, add <section class = "home" id = "home"> </section> to create an area for the home section of the website

2. In this new section tag, add <nav> </nav> to create an area for the navigation bar

3. In the nav tag, add <img src = "images/logo.png> to place the logo image

4. In the nav tag, add <ul> </ul> to create an unordered list area for the tab links

5. Add all of the following tab links in the unordered list tag:

<a href = "#home"> <li> Home</li></a>
<a href = "#about"> <li> About</li></a>
<a href = "#photos"> <li> Photos</li></a>

6. Outside the nav tag, add <h1>Michelle Flores</h1> to display the name

7. Add <br> </br> after the name tag to create a break or spacing on the page

8. Below the name tag, add <h2> Photographer</h2> to display the job title

9. Add <br> </br> after the name tag to create a break or spacing on the page

10. Below the job tag, add <a href="mailto:michelle_flores@gmail.com" class = "email">michelle_flores@gmail.com</a> to display the email address and make it take the user directly to their mailbox

5

Style Home Section


In the style.css file, you will format all of the content in the home section.

1. Get rid of the extra white space in the margins by adding the following block:

* {
margin: 0;
padding: 0;
}

2. Set the scroll behavior of the website to smooth by adding the following block:

html {
scroll-behavior: smooth;
}

3. For the header section, set the minimum height to 600 pixels, put the background image to the photographer image with a linear gradient, adjust the background size to cover the entire header, set the background position to the center, filter the background image with a contrast of 1.25, add 100 pixels of bottom margining, add 15 pixels of padding all on sides, and add an additional 30 pixels of padding to the left. The block of code should look like the following:

.home {
min-height: 600px;
background-image: linear-gradient(rgba(255, 255, 255, 0.559),
rgba(255, 255, 255, 0.793)), url(images/photographer.jpeg);
background-size: cover;
background-position: center;
filter: contrast(1.5);
margin-bottom: 100px;
padding: 15px;
padding-left: 30px;
}

4. Display all elements in the navigation bar horizontally and in line with each other by adding the following block:

.home nav {
display: flex;
align-items: center;
}

5. Set the width of the logo to 75 pixels and add 15 pixels of margin to the right by adding the following block:

.home nav img {
width: 75px;
margin-right: 10px;
}

6. For each individual tab link, set the list style type to none, adjust the display to flex in line with each other, create a padding area of 26 pixels by 30 pixels, set the text color to black, adjust the text decoration to none, set the font size to 20 pixels, adjust the font family to sans-serif, and add a 0.5 second transition for when the user stops hovering their mouse over the tab. The block of code should look like the following:

.home nav ul a li {
list-style-type: none;
display: inline-flex;
padding: 26px 30px;
color: rgb(0, 0, 0);
text-decoration: none;
font-size: 20px;
font-family: sans-serif;
transition: 0.5s;
}

7. When the user hovers over a tab link, set it to where the background color changes to black while the text color changes to white within 0.5 seconds. The block of code should look like the following:

.home nav ul a li:hover {
background-color: rgb(0, 0, 0);
color: rgba(255, 255, 255, 0.92);
transition: 0.5s;
}

8. For the name tag, set the font size to 70 pixels, adjust the font family to sans-serif, and add 18 pixels of margin on the top. The block of code should look like the following:

.home h1 {
font-size: 70px;
font-family: sans-serif;
margin-top: 180px;
}

9. For the job tag, set the font size to 40 pixels, adjust the font family to sans-serif, and make the text color gray. The block of code should look like the following:

.home h2 {
font-size: 40px;
font-family: sans-serif;
color: gray;
}

10. For the email tag, set the font size to 20 pixels, adjust the font family to sans-serif, and make the text color gray. The block of code should look like the following:

.home .email {
font-size: 20px;
font-family: sans-serif;
color: gray;
}

6

Create Framework of About Section


In the index.html file, you will create the framework of the about section of the website where the user can read a brief description of the author.

1. In the body tag, add <section class = "about" id = "about"> </section> to create an area for the about section of the website

2. In this new section tag, add <h1>About </h1> to display the section title

3. Below this title tag, add <p>lorem </p> to display the description.

4. Place your cursor at the end of lorem and click TAB to create placeholder text

7

Style About Section


In the style.css file, you will format all of the content in the about section.

1. For the about section, set the minimum height to 400 pixels, add 15 pixels of padding on all sides, and add an additional 30 pixels of padding to the left. The block of code should look like the following:

.about {
min-height: 400px;
padding-left: 30px;
padding: 15px;
}


2. For the section title, set the font size to 40 pixels and adjust the font family to sans-serif. The block of code should look like the following:

.about h1 {
font-size: 40px;
font-family: sans-serif;
}


3. For the description text, set the font size to 20 pixels, adjust the font family to serif, and add 20 pixels of margin on the top. The block of code should look like the following:

.about p {
font-size: 20px;
font-family: serif;
margin-top: 20px;
}

8

Create Framework of Photos Section


In the index.html file, you will create the framework of the photos section of the website where the user can view photos that the author has captured.

1. In the body tag, add <section class = "photos" id = "photos"> </section> to create an area for the about section of the website

2. In this new section tag, add <h1> Photos</h1> to display the section title

3. Below the section title, add <div class = "collage"> Photos</div> to create an area for the photos

4. In this new div tag, add all of the following images:

<img src = "images/photo1.jpg" class = "photo1">
<img src = "images/photo2.jpg" class = "photo2">
<img src = "images/photo3.jpg" class = "photo3">

9

Style Photos Section


In the style.css file, you will format all of the content in the photos section.

1. The photos section will be formatted exactly like the about section. To avoid repeating the same steps you took to style the about section, you can instead include the photos section in the about section styling block, which should look like the following:

.about, .photos {
min-height: 400px;
padding-left: 30px;
padding: 15px;
}


2. You will format the title for the photos section exactly like the about section. Similarly to the previous step, include the photos title in the about title styling block, which should look like the following:

.about h1, .photos h1 {
font-size: 40px;
font-family: sans-serif;
}


3. For the photo collage area, set the display to flex and make it flex-wrap so that the images will line up horizontally and adjust to the size of the screen. Also, add 20 pixels of margin on the top. The block of code should look like the following:

.photos .collage {
display: flex;
flex-wrap: wrap;
margin-top: 20px;
}


4. For each individual image, add 20 pixels of padding to the right and bottom and set the width to 500 pixels. The block of the code should look like the following:

.photos .collage img {
padding-right: 20px;
padding-bottom: 20px;
width: 500px;
}

10

Create Responsive Design


In the style.css file, you will create styling conditions for when the user’s screen size is smaller than a normal computer screen. Making styling conditions for mobile devices or any small screen device is known as responsive design.

1. Create a media query where you will make styling changes when the width of the user’s screen is less than 700 pixels. The query will look like the following:

@media (max-width: 700px) {

}


2. Inside of this media query, get rid of the navigation bar display and position all of the text and images of the website in the center. The blocks of code will include the following:

.home nav ul {
display: none;
}
html {
text-align: center;
padding: 15px;
}
.photos .collage {
display: flex;
justify-content: center;
}


3. Set the name text size to 50 pixels, the job text to 30 pixels, and about description to 16 pixels. The blocks of code will include the following:

.home h1 {
font-size: 50px;
}
.home h2 {
font-size: 30px;
}
.about p {
font-size: 16px;
}

11

Host Website on GitHub


You will need to create an account in GitHub so that you can publish your website. Although Replit can also be used to publish websites for free, the domain names are not as clean and customizable as GitHub.

1. Go to github.com

2. Click Sign Up

3. Follow the instructions to finish creating your account

4. On the home dashboard, click Create New Repository

5. Name the repository michelleflores

6. Click Create

7. On the repository page, click Upload An Existing File

8. Upload the index.html file, style.css file, and images folder. You should already have the images folder downloaded. To download the html and css files, go back to Replit and in the file panel to the left, click on the three dots next to each file and click Download

9. Click Commit Changes

10. On the repository page, click Settingsand then Pages

11. Under build and deployment, select the Main branch and then click Save

12. Wait a few minutes and then refresh the page. At the top of the screen under GitHub Pages, you should see your newly made URL to the website portfolio: https://{username}.github.io/michelleflores
